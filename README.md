# Task 3 - Dashboard Design

## Objective
Design an interactive dashboard for business stakeholders using Power BI.

---

## Project Name
FinSight Analytics Dashboard

---

## Tools Used
- Power BI
- Excel
- Kaggle Dataset

---

## Dataset
Global Superstore Sales Dataset

The dataset contains sales and financial data across multiple countries, categories, and customer segments.

---

## Dashboard Features
- KPI Cards
- Global Sales Map
- Sales Trend Analysis
- Profit by Category
- Sales by Segment
- Interactive Slicers

---

## Key Metrics
- Total Sales: 13M
- Total Profit: 1.47M
- Profit Margin: 11.64%
- Total Quantity: 178K

---

## Key Insights
- Sales increased consistently over time
- Technology generated highest profit
- Consumer segment contributed highest sales
- APAC and EU showed strong performance

---

## Outcome
This dashboard helps business stakeholders monitor performance, identify growth opportunities, and make data-driven decisions.
